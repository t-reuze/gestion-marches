4.3 — Autres marchés invest

Pour tout marché de mockData.js avec secteur === 'investissements' qui n'est pas explicitement listé ci-dessus, une config par défaut est générée :

excelMarcheLabel: "<Nom marché>_<Référence>"
ppeRef:           "<Référence>"  (ex: PPE033, AL109)
lotLabel:         (n) => `<Nom marché>_<Référence>_Lot${n}`
typeEquipementByLot:    {} // vide → saisie manuelle ou apprise depuis BDD
consommablesLotRange:   null

Les conventions sont aussi enrichies par l'apprentissage automatique depuis la BDD (voir doc Algorithme §7). Marchés concernés : inv-accelerateurs, lert, med-nuc, ctrl-regl-al109, ctrl-qual-al083, rad-int, ctrl-regl-al080, telerad, ia, pacs-dacs.
